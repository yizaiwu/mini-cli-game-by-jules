# 文字冒險遊戲 (CLI Text Adventure)

這是一個使用 Python 打造的輕量級、指令列（CLI）文字冒險遊戲。專為喜愛經典 RPG 風格的玩家設計，提供一個單人、但具備 MMO 風格的遊戲體驗。

## 遊戲特色

- **單人 MMO 風格體驗**：享受一個廣闊、可持續探索的單人世界。
- **繁體中文介面**：所有遊戲訊息與描述皆以繁體中文呈現，指令則使用簡單的英文（如 `go north`）。
- **網格地圖探索**：在一個 3x3 的網格世界中移動，探索不同的地點。
- **即時戰鬥系統**：採用多執行緒（threading）技術，實現了玩家與怪物之間的自動攻擊，同時允許玩家輸入指令使用特殊技能。
- **角色成長**：玩家角色擁有生命值、攻擊力、防禦力等屬性，可透過戰鬥獲得經驗值並升級。
- **怪物與戰利品**：目前遊戲中實作了「史萊姆」作為初始怪物，擊敗牠可以獲得經驗值。
- **遊戲結束**：當玩家生命值歸零時，遊戲將會結束。

## 如何開始

### 1. 環境需求

請確保您的電腦已安裝 **Python 3**。若尚未安裝，可至 [Python 官方網站](https://www.python.org/) 下載。

*在 Windows 上安裝時，請務必勾選 `Add Python to PATH` 的選項。*

### 2. 執行遊戲

1.  將本專案的檔案下載或複製到您的電腦上。
2.  打開您的終端機（命令提示字元）。
3.  使用 `cd` 指令切換到專案的根目錄下（您應該能看到 `main.py` 檔案）。
4.  執行以下指令來啟動遊戲：

    ```bash
    python main.py
    ```

## 專案結構

本專案採用了模組化的程式碼結構，以提高可讀性與未來的可擴充性。主要邏輯被拆分至 `game` 目錄下的不同檔案中：

- `player.py`: 處理玩家角色相關的邏輯。
- `monster.py`: 處理怪物相關的邏輯。
- `world.py`: 存放遊戲世界與地圖的資料。
- `combat.py`: 負責處理核心的戰鬥系統。
- `main.py`: 遊戲的主進入點與遊戲迴圈。
